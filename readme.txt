网站蜘蛛/访客记录，主要包含了各大搜索引擎的蜘蛛访问记录，以及网站的访问流量进行监控。


== Description ==

本插件会记录网站蜘蛛和访客的访问。

1、开启插件以后，如果有百度、谷歌、360、搜狗、神马、必应、头条蜘蛛访问网站就会被统计。无需在查看网站日志，是比较省心的工具。

2、网站蜘蛛/访客记录可以按日期查询也可以按搜索引擎查询，并且注重了404、200记录的蜘蛛状态。

3、流量监控是记录访客的浏览量（PV）、访客量（UV）、IP值，以及来源网站排名、新老访客和地域分布。

== Installation ==

方式1：在线安装(推荐)
1. 进入WordPress仪表盘，点击“插件-安装插件”，关键词搜索“网站蜘蛛/访客记录”，找搜索结果中找到“蜘蛛-关键字”插件，点击“现在安装”；
2. 安装完毕后，启用 `蜘蛛-关键字` 插件.
3. 通过“设置”->“网站蜘蛛/访客记录” 进入插件设置界面.
4. 至此，该插件安装完毕，即可访问网站蜘蛛/访客记录。

方式2：上传安装

FTP上传安装
1. 解压插件压缩包key_spider.zip，将解压获得文件夹上传至wordpress安装目录下的 `/wp-content/plugins/`目录.
2. 访问WordPress仪表盘，进入“插件”-“已安装插件”，在插件列表中找到“网站蜘蛛/访客记录”，点击“启用”.
3. 通过“设置”->“网站蜘蛛/访客记录” 进入插件设置界面.
4. 至此，该插件安装完毕。

仪表盘上传安装
1. 进入WordPress仪表盘，点击“插件-安装插件”；
2. 点击界面左上方的“上传按钮”，选择本地提前下载好的插件压缩包key_spider.zip，点击“现在安装”；
3. 安装完毕后，启用 `网站蜘蛛/访客记录` 插件；
4. 通过“设置”->“网站蜘蛛/访客记录” 进入插件设置界面.
5. 至此，该插件安装完毕，即可访问百度站长后台查看网站蜘蛛/访客记录。
